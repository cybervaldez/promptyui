Key Differences

  • A (Excel Border) — Most explicit boundary. Clear visual "window frame" around reachable values. Adds DOM nesting.
  • B (Opacity Fade) — Lightest touch. No extra DOM, just CSS opacity. Out-of-window values still visible as chips.
  • C (Bracket + Dot-List) — Most compact. Out-of-window values take less space as text. Clearest hierarchy separation.
  • All three: chips are read-only (no pin/toggle). Active = composition-derived, interactive selection stays in block dropdowns.
